# orderno

NPM package for creating distinct order numbers in a high volume distributed system.

[![NPM Version][npm-image]][npm-url]
![TEST][https://github.com/jluterek/orderno/workflows/test/badge.svg]

### Installing

```bash
npm i -S orderno
```

## Running the tests

```bash
npm run test
```

### Coding Style

```bash
npm run lint
npm run format
```
